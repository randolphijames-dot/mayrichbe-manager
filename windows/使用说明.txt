  Mayrichbe Manager - Windows 使用说明
  INS/YT 矩阵管理系统
═══════════════════════════════════════════

【前提条件】
  - Windows 10 或以上
  - 安装 Python 3.10+（https://www.python.org/downloads/）
    ⚠ 安装 Python 时务必勾选 "Add Python to PATH"

【首次使用（只需一次）】
  1. 解压整个 ZIP 文件到任意目录（如桌面）
  2. 双击「安装.bat」→ 等待自动安装依赖（2-3 分钟）
  3. 看到 "安装完成" 后关闭窗口

【每次使用】
  1. 双击「启动.bat」
  2. 等待 3 秒，浏览器会自动打开 http://localhost:8000
  3. 在浏览器里操作即可
  4. 用完后关闭 cmd 窗口即可停止服务

【功能说明】
  - 账号管理：添加 INS/YT 账号，支持 CSV 批量导入
  - 素材库：上传视频/图片，分配到对应账号
  - 定时发布：设定发布时间，系统自动发送
  - 批量改资料：批量修改简介和头像
  - 智能养号：自动刷 Feed、点赞，模拟真人活跃
  - 消息中心：集中查看/回复所有账号的评论和私信
  - 自动截流：通过话题/对标账号自动互动引流

【常见问题】
  Q: 双击安装.bat 没反应？
  A: 右键 → "以管理员身份运行"

  Q: 提示"python 不是内部命令"？
  A: Python 安装时没勾选 "Add to PATH"，重新安装并勾选

  Q: 启动后浏览器打不开？
  A: 手动在浏览器地址栏输入 http://localhost:8000

  Q: 关掉 cmd 窗口后还能定时发布吗？
  A: 不能。cmd 窗口就是服务进程，关掉 = 停止服务。
     建议不用时最小化窗口，不要关闭。
