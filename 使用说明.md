# Mayrichbe Manager 使用说明

## 📦 给朋友的安装包

### Windows 版本

**包含文件**：
```
Mayrichbe-Manager/
  ├── START_WINDOWS.bat        ← 双击这个启动
  ├── backend/                  ← 后端代码（不要动）
  ├── README.md                 ← 详细文档
  └── 使用说明.md               ← 本文件
```

**安装步骤**：

1. **安装 Python 3.11+**
   - 下载：https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"

2. **安装 AdsPower**（Instagram 发布需要）
   - 下载：https://www.adspower.net/
   - 安装并运行

3. **启动应用**
   - 双击 `START_WINDOWS.bat`
   - 首次启动会自动安装依赖（约 2-3 分钟）
   - 以后启动只需 5 秒
   - 浏览器会自动打开 http://localhost:8000

4. **使用**
   - 添加账号（Instagram/YouTube）
   - 上传素材（视频/图片）
   - 设置定时发布
   - 查看发布日志

---

### Mac 版本

**包含文件**：
```
Mayrichbe-Manager/
  ├── START_MAC.command         ← 双击这个启动
  ├── backend/                  ← 后端代码（不要动）
  ├── README.md
  └── 使用说明.md
```

**安装步骤**：

1. **安装 Python 3.11+**
   ```bash
   brew install python@3.11
   ```

2. **安装 AdsPower**
   - 下载：https://www.adspower.net/
   - 安装并运行

3. **启动应用**
   - 双击 `START_MAC.command`
   - 如果提示权限问题，右键 → 打开
   - 首次启动会自动安装依赖
   - 浏览器会自动打开

## ❓ 常见问题

### Q1：双击启动脚本没反应？
**A1**：检查是否安装了 Python
- Windows: 打开 CMD，输入 `python --version`
- Mac: 打开终端，输入 `python3 --version`
- 如果提示"命令不存在"，需要先安装 Python

### Q2：浏览器显示"无法访问"？
**A2**：等待 10-20 秒，后端服务正在启动

### Q3：Instagram 发布失败？
**A3**：确认 AdsPower 是否正在运行
- 打开 AdsPower 应用
- 确保浏览器 Profile 已创建

### Q4：如何关闭应用？
**A4**：
- Windows: 关闭黑色窗口（CMD）
- Mac: 在终端按 Ctrl+C

### Q5：如何在其他电脑使用？
**A5**：复制整个文件夹到新电脑，重新执行安装步骤

## 🔄 更新方法

1. 备份旧版本的 `backend/social_manager.db`（数据库）
2. 下载新版本覆盖
3. 恢复数据库文件
4. 重新启动
